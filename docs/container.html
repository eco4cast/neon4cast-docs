Appendix B — Docker Container

We provide a R-focused docker container that can be used in GitHub Actions or other workflows for generating forecasts.

The container eco4cast/rocker-neon4cast can be found on Docker Hub (https://hub.docker.com/repository/docker/eco4cast/rocker-neon4cast). eco4cast/rocker-binder also avialable and is the same as eco4cast/rocker-neon4cast but minor differences that are specific for using as a Binder. Both containers are built off the rocker/geospatial:latest image and include Rstudio.

Follow the directions at https://rocker-project.org for launching the container locally from your web browser (requires a local installation of Docker Desktop)
